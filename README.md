# Package: @decohome/ui

React UI Toolkit Package composed by:

- 28 Components.<br>
- 14 Utilities.<br>
- 23 Icons.<br>
- 10 Hooks.<br>
<hr>

### Prerequisites

- Node.js [v16.19.1](https://nodejs.org/dist/v16.19.1/).
- npm 8.19.3.
- Visual Studio Code.
<hr>

### Installation

```sh
npm run install
```

<hr>

### Build

```sh
npx nx build ui
```

This will generate dist folder and other files in the root folder.

<hr>

### Remix Integration

<br>


1 - Install @decohome/ui
<br>

```sh
yarn install
```

2 - start Project
<br>

```sh
npx nx run backoffice:dev:development --skip-nx-cache

```
